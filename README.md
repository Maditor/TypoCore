
## ✨ Tính năng của TypoCore

TypoCore là panel CEP giúp tăng tốc quá trình typesetting webtoon với các công cụ đơn giản, dễ dùng và hiệu quả.

### 🔹 Quick Layout
- Xem nhanh nhiều bố cục bóng thoại chỉ với 1 lần load  
- Hỗ trợ 2 chế độ ngang và dọc tiết kiệm không gian
- Giúp sắp xếp thoại nhanh và gọn hơn  

### 🔹 Chia dòng thông minh
- Tự động chia text thành **2 / 3 / 4 dòng**  
- Giữ bố cục cân đối, dễ đọc  

### 🔹 Điều chỉnh Typography
- Tăng giảm nhanh **font size & leading**  
- Tinh chỉnh khoảng cách chữ dễ dàng  

### 🔹 Công cụ nhanh
- Resize, căn giữa text  
- Group, select, check layer  
- Copy / Paste hiệu ứng (FX)  

### 🔹 Logo Tool
- Chọn logo trong setting  
- Dán logo vào **tất cả ảnh đang mở**  

### 🔹 Tùy chỉnh panel
- Ẩn / hiện các tool không cần thiết  
- Thay đổi Load thoại Tự động - Thủ Công
---

TypoCore tập trung vào **tốc độ, sự đơn giản và tính thực tế**, phù hợp cho workflow typesetting webtoon.

### 🔹 Hướng Dẫn Cài Đặt

- Tải và giải nén file .zip  
- Mở thư mục vừa giải nén  
- Tắt Photoshop (nếu đang mở)  
- Chạy file `Install_win.bat`  
- Nhấn Enter và chờ quá trình cài đặt hoàn tất  
- Nhấn Enter lần nữa để thoát  

- Mở Photoshop  
→ Vào: **Window → Extensions → TypoCore**
